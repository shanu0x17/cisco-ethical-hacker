# Cisco Ethical Hacker
---
## Overview

This repository documents my practical learning while completing the Cisco Ethical Hacker course.

It contains original notes, hands-on laboratory exercises, scripts, and cheat sheets created throughout the learning process.

The purpose of this repository is to reinforce concepts through practical implementation and maintain a structured cybersecurity knowledge base.


## Objectives

• Build a strong understanding of ethical hacking concepts

• Gain hands-on experience with common offensive security tools

• Document practical labs and findings

• Develop reusable scripts for automation

• Maintain a structured cybersecurity portfolio


## Repository Structure

Personal learning notes.

labs/
Hands-on exercises and walkthroughs.

Automation written in Python and Bash.
