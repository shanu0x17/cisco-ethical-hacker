# Lab 01 – Investigate Kali Linux

## Objective

Become familiar with the Kali Linux graphical interface (GUI), terminal environment, and basic Linux commands used during penetration testing.

---

## Environment

- Operating System: Kali Linux
- Platform: Cisco Ethical Hacker Virtual Lab
- Interface: GUI & Terminal

---

## Activities Performed

- Explored the Kali Linux desktop environment.
- Navigated the Applications menu and reviewed tool categories.
- Customized and explored the top panel settings.
- Opened and used the Linux terminal.
- Used the `man` command to access command documentation.
- Reviewed commonly used Linux commands.

---

## Commands Observed

```bash
man man
pwd
mv
chmod
chown
dd
ps
su
```

---

## Key Learnings

- Kali Linux is a Debian-based distribution designed specifically for security testing.
- The Applications menu organizes tools by category, making navigation easier.
- The terminal is the primary interface for many penetration testing tasks.
- The `man` command provides built-in documentation for Linux commands.
- Understanding basic Linux commands is essential before using advanced security tools.

---

## Skills Practiced

- Kali Linux Navigation
- Linux Terminal Basics
- Command Documentation
- System Familiarization

## Personal Reflection

This lab helped me become comfortable with the Kali Linux environment. I learned how to locate security tools, navigate the desktop, and use the terminal to access command documentation, which will be useful throughout the course.
